# Upcoming Events & Timetables

Headcode includes an **Upcoming Events** tool that lets you pull in event packs (such as galas) and their related timetables.

This is designed so you can quickly bring event information into your app without creating every item manually.

## Where to find it

1. Open any main screen in Headcode
2. Tap the **Tools** icon in the top bar
3. Tap **Upcoming Events**

You can also open **Timetables** from the same Tools menu to view timetables you have already imported.

## How it works (simple view)

When you open Upcoming Events, Headcode:

- Checks the latest official event list
- Shows available events as cards
- Lets you import the event and, when available, its timetable

Each event card shows a clear action such as:

- **Import Event** (first time)
- **Update Event** (if a newer version is available)
- **Imported** (already up to date)

## Importing an event

Tap **Import Event** on the event you want.

Once imported, the event becomes available in your app data, including event pickers when logging or editing spots.

## Importing a timetable

If a timetable is available for that event, you can tap **Import Timetable** from the same event card.

Imported timetables appear in the **Timetables** screen (from the Tools menu).

## Keeping data current

- Tap the **Refresh** icon on the Upcoming Events screen to check for updates
- If an update is available, Headcode shows **Update Event** or **Update Timetable**
- You can choose when to apply updates

## If you edit imported data

If you manually change an imported event or timetable, Headcode treats it as your local version so your edits are not unexpectedly replaced.

## If you are offline

If your connection is unavailable, Headcode may still show the most recently loaded event list until you are online again.
