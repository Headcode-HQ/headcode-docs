# Locomotive Types

Locomotive Types (also called traction types) describe how a locomotive is powered. These are used when recording locomotive details and help you filter and analyse your spots by traction.

<!-- Screenshot: The Locomotive Types list showing default types -->

## Default Types

Headcode includes a set of common traction types to get you started:

- Diesel
- Electric
- Steam
- DMU (Diesel Multiple Unit)
- EMU (Electric Multiple Unit)

## Adding a New Type

1. Go to **Settings > Reference Data > Locomotive Types**
2. Tap the **+** button
3. Enter the type name
4. Tap **Save**

## Editing a Type

1. Find the type in the list
2. Tap the **pencil icon**
3. Update the name
4. Tap **Save**

::: info
Editing a type name does **not** update existing spots that used the old name. Existing spots retain the value that was recorded at the time.
:::

## Deleting a Type

1. Find the type in the list
2. Tap the **bin icon**
3. Confirm the deletion

Deleting a type removes it from the suggestions list but does not affect existing spots.
