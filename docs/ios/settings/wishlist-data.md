# Wishlist Data

The Wishlist Data screen lets you view a summary of your active Wish List and permanently delete all wish list entries.

<!-- Screenshot: The Wishlist Data screen showing active wish count and delete option -->

## Wish List Summary

At the top of the screen you'll see the number of **active wishes** — locomotives on your list that you haven't spotted yet.

## Deleting All Wish List Items

::: danger
This action is permanent. Deleted wish list entries cannot be recovered unless you have a [Full Backup](/ios/settings/import-export/export-full).
:::

To delete all wish list items:

1. Go to **Settings > Wishlist Data**
2. Tap **Delete All Wish List Items** in the Danger Zone section
3. Confirm the deletion when prompted

All active wishes will be removed. This does not affect your logged spots.
