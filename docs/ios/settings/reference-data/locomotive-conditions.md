# Locomotive Conditions

Locomotive Conditions describe the physical state or operational status of a locomotive at the time you spotted it. These help you record and filter spots by how a loco looked or where it was.

<!-- Screenshot: The Locomotive Conditions list showing default conditions -->

## Default Conditions

Headcode includes a set of common condition descriptors:

- Pristine
- Good
- Weathered
- Stored
- Heritage

## Adding a New Condition

1. Go to **Settings > Reference Data > Locomotive Conditions**
2. Tap the **+** button
3. Enter the condition name
4. Tap **Save**

## Editing a Condition

1. Find the condition in the list
2. Tap the **pencil icon**
3. Update the name
4. Tap **Save**

::: info
Editing a condition name does **not** update existing spots that used the old name. Existing spots retain the value that was recorded at the time.
:::

## Deleting a Condition

1. Find the condition in the list
2. Tap the **bin icon**
3. Confirm the deletion

Deleting a condition removes it from the suggestions list but does not affect existing spots.
